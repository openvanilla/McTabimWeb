使用方式

小麥他命是一套查表式輸入法，可以使用倉頡、速成、大易、行列等輸入法。直接輸入字根後，就會顯示可用的候選字，再按下選字鍵即可完成輸入。

特殊按鍵如下：

  • 按下 ` 之後，進入符號輸入模式，然後按下鍵盤上的其他按鍵，就會顯示全形符號。
  • 按下 `M 或 ``` 之後，可以叫出功能選單。

打出某些字詞之後，可以選擇聯想詞。輸入方式是按下 Shift + 數字按鍵。
