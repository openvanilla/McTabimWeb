軟體說明

小麥他命是一套以網頁技術打造的查表式輸入法，可以使用倉頡、速成、大易、行列等。網頁版本方便在無法自行安裝輸入法的環境下，依然可以有順手的輸入法，像是某些限制較大的平板與電子紙裝置。

除了網頁版本外，也提供 ChromeOS 版本，也可以透過 PIME 在 Windows 上使用。

使用方式

直接輸入字根後，就會顯示可用的候選字，再按下選字鍵即可完成輸入。

特殊按鍵如下：

  • 如果按下 Ctrl 加上符號按鍵，可以快速輸出常用全形標點符號，例如問號、句號等。
  • 按下 ` 之後，進入符號輸入模式，然後按下鍵盤上的其他按鍵，就會顯示全形符號。
  • 按下 `M 或 ```（連按三下`）之後，可以叫出功能選單。
  • 啟用萬用字元選項後，可以先輸入字根，再輸入 * 來查詢不確定的部首。
  • 啟用同音字反查選項後，可以先打一個字，再用 ` 來查詢同音字。

打出某些字詞之後，可以選擇聯想詞。輸入方式是按下 Shift + 數字按鍵。

吳守禮臺語注音

小麥他命輸入法使用的吳守禮臺語注音表格，來自 OpenVanilla 專案。 是由吳彥勳 yenhsunwu@gmail.com 參考吳守禮教授編著的《新編台灣閩南語用字彙編》繕打而成。使用時建議搭配臺語字體一起使用：

  • 臺語字體下載網址
  • 臺語字音資料網址

本輸入法使用對應鍵盤位置修改自「華音延伸鍵盤」，

在原本的鍵盤設計中，將聲調定義在 F1-F8 等按鍵上，在一般電 腦上並不方便輸入。將聲調符號位置稍作修改：

  • 第一、二、三、五聲對應華音一、四、三、二聲輸入鍵
  • 第四聲為 $，第八聲為 ^
  • 第六聲與第二聲相同
  • 第七聲為 7，輕聲為 &

通用拼音客家話

本專案提供通用拼音客家話海陸腔與四縣腔輸入表格。原始版權資訊如下：

本檔是由陳念波 (nienpo@yahoo.com) 依據《台灣客家話辭典》 編者徐兆泉老師提供的同音字檢索資料使用 python script 製作並 加入「四海話」發音本檔案依據Creative Commons 依「姓名標 示」、「相同方式分享」、「禁止商業使用」授權，歡迎各界散佈及 使用。授權詳情請參閱 http://creativecommons.org/licenses/by-nc-sa/2.5/tw/deed.zh_TW

客家話裏的「我」是用「亻厓」這個字，但此字的 unicode 編碼 是 U+2028E，是十六進位的五位數，屬於 CJK Unified Ideographs Extension B,含有此字的字型檔不是每個人都有。此 cin 檔含此 字。但是，為了方便一般使用狀況，把「亻厓」這個字的發音nga̠i (四縣腔) ngai (海陸腔) 對應到「我」這個字。此檔含有 11 個 Privat Usage Area 編碼段 (U+E000 to U+F8FF) 的字，目前沒有收 錄於unicode 中，是徐老師造的字。本檔另外以示意方式表示這些 字，如 (亻恩)，並加入 LaTeX 組合# 這些字的指令 (使用 LaTeX 時可用)，應不會造成一般使用者的困擾。
